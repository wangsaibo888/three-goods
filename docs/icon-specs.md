# 图标设计规范

## TabBar 图标
- 尺寸：81px × 81px (3倍图)
- 格式：PNG
- 颜色规范：
  - 未选中状态：#999999
  - 选中状态：#1AAD19
- 图标列表：
  1. home.png / home_selected.png
  2. record.png / record_selected.png
  3. calendar.png / calendar_selected.png
  4. list.png / list_selected.png

   - 尺寸：60px × 60px
   - 颜色：#999999
